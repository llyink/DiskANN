3. Recall_Streaming
max_points:        The total size of the dataset, here is set to 1000000
insert_percentage: Percentage of inserted points, decimal input
build_percentage:  Percentage of build points, decimal input
delete_percentage: Percentage of delete points, decimal input

**build with 100% points of 1,000,000 points, insert n% points, delete n% points (you can replace n with any number in [0,100])
